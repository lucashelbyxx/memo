泛型是一种参数化类型特性

使用泛型好处

1. 编译期没有警告，运行期间不会出现ClassCastException
2. 不用强转
3. 复用

比如说创建一个ArrayList，把A类对象存进去，但我取出来的时候想要B类对象，强转成A对象就会报类型转换异常。



泛型擦除，通过桥接方式强转回我们需要的类型

在类常量池里面保留泛型信息
